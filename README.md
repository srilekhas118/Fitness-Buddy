🤸‍♀️Fitness Buddy — Agentic AI Health Coach

## 💻 Project Overview
This repository contains the official main project submission for the **IBM SkillsBuild Advanced AI Program**. "Fitness Buddy" is a conversational, autonomous Agentic AI application built using **IBM Cloud Lite services** and **watsonx Orchestrate** [📡]. It guides users toward healthier lifestyles by automating multi-step workflows.

## 🏗️ System Architecture
The application uses a **Router Agentic AI Design Pattern**. When a user types a query, the system processes the input variable using conditional branching nodes inside a visual orchestration canvas to autonomously route the text response without manual intervention.



## 🚀 Core Features
- **Home Workouts Node**: Displays a structured 15-minute bodyweight routine (squats, push-ups, planks, jumping jacks).
- **Nutritious Meals Node**: Suggests healthy, simple meal ideas for breakfast, lunch, and dinner.
- **Daily Motivation Node**: Provides encouraging, habit-building psychological cues to reinforce consistency.
